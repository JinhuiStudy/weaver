## Description (260자 이하)

Weaver turns every AI agent into a public, forkable, rate-able artifact. One click forks a stranger's prompt into your workspace. Nightly mutations let top agents evolve. Run on Cloudflare's free tier — you pay $0, we pay $0.

## First comment (maker)

Hey everyone 👋

I'm Jinhui, solo dev in Seoul. I built Weaver in 14 weeks because I wanted agents to feel like **remixable public media** — the same way CodePen made CSS experiments contagious.

Three principles drove the design:

1. **Fork + Genealogy 1등 시민** — every public agent has an `@handle/slug` URL, a one-click fork button, and a family tree.
2. **$0 everywhere** — runs on Cloudflare Workers + D1 + Workers AI. No user is ever asked for a credit card. No API key required for the default flow.
3. **Auto-evolution** — 5 prompt mutations nightly, shadow-evaluated with a Llama judge, surfaced to creators as "🧬 v2 · 17% better" banners.

Everything is Apache-2.0, OTEL-traced, and has 450+ tests.

Playground: https://weaver-web.jinhuistudy.workers.dev  
Repo: https://github.com/JinhuiStudy/weaver  
Docs: https://weaver-web.jinhuistudy.workers.dev/docs

Would especially love feedback on the **fork-and-evolve loop** vs familiar prompt-library patterns. 🙏
